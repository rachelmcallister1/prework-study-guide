HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user
  • HTMl is made up of elements to give content and add a basic framework of a page. It uses opening and closing tags.

CSS

  • CSS stands for Cascading Style Sheets
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicaes how much space we want around the content inside of an element.
  • CSS defines the style. Such as colors, aesthetics, visual layout, and organization.
  • When a developer wants to style an element directly in the HTML file, they use inline CSS.

Git

  • git status: checks what brand we are on.
  • git checkout: switches to a new branch and checks it into the working directory.
  • git checkout -b branch-name: creates a new branch and swtiches to it.
  • git add -A: this command adds modifications in the current working branch to the staging area, then indicates to add all changes.

JavaScript

  • In web development, JavaScript is a tool that takes the static HTML/CSS webpage and makes it interactive. Meaning, everytime you click a button, retrieve data, or fill out a form, JavaScript is at work!
  • JavaScript is case sensitive.
  • Functions are the builing blocks of JavaScript.
  • As a dynamic language, variables are not directly associated with any particular value type, and any variable can be assigned and re-assigned values of all types.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
